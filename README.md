# Fourier Transform and Circuit Simulation

Course – Practical Assignment  
University of Pretoria  
Completed: October 2020

---

## Project Overview

This project focuses on analysing and approximating pulse train signals using Fourier transforms and evaluating circuit responses in both time and frequency domains. 

The work involves deriving mathematical expressions for signals, applying Fourier and Laplace transforms, and analysing system responses of RC, RL, and RLC circuits. Theoretical results are compared with simulated outputs to evaluate accuracy.

Simulations are performed to observe how different circuit configurations and parameter changes affect system behaviour, including damping conditions and frequency responses.

---

## Objectives

- Approximate a pulse train using the Fourier transform
- Graphically represent signals in the frequency domain
- Analyse and derive system responses using Laplace transforms
- Simulate RC, RL, and RLC circuits
- Compare theoretical results with simulated circuit behaviour
